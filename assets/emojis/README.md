# Emoji Assets

Эта папка содержит анимированные GIF смайлы для Vtalk мессенджера.

## Требуемые файлы:
- smile.gif - Улыбка
- cool.gif - Крутой
- shock.gif - Шок
- tongue.gif - Показывает язык
- heart.gif - Сердце
- thumbsup.gif - Палец вверх
- fire.gif - Огонь
- star.gif - Звезда

## Размеры:
- Рекомендуемый размер: 32x32 пикселя
- Максимальный размер файла: 50KB
- Формат: GIF с анимацией

## Источники для загрузки:
Можно использовать бесплатные GIF смайлы с:
- Giphy
- Tenor
- Imgur
- Или создать свои в Photoshop/After Effects
