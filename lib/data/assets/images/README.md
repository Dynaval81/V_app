# VTalk Logo Placeholder

This folder will contain the VTalk logo files:

- vtalk_logo.png - Main logo for headers (120x40px recommended)
- vtalk_logo_dark.png - Dark theme variant (optional)
- app_icon.png - App icon (1024x1024px)

Once you add your logo files, they will be automatically used in the app headers.

## Instructions:
1. Add your logo files to this folder
2. Make sure they're in PNG format with transparent background
3. Recommended sizes:
   - Header logo: 120x40px
   - App icon: 1024x1024px
4. Run `flutter pub get` to refresh assets
